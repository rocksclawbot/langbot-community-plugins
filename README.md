# LangBot Community Plugins

A collection of practical plugins for [LangBot](https://github.com/langbot-app/LangBot).

| Plugin | Description |
|--------|-------------|
| [tts-emotion-router](./tts-emotion-router/) | Route TTS output to different voices based on detected emotion |
| [disaster-alert](./disaster-alert/) | Real-time earthquake & weather alerts pushed to chat |

## Installation

In LangBot WebUI → Plugin Marketplace → click "+" → paste the GitHub URL of the plugin folder.

Or copy the plugin folder to `data/plugins/` and restart the runtime container.
