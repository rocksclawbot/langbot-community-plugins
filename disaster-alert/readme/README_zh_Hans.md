# 灾害预警推送

实时地震、海啸和气象预警推送到你的聊天群组，不错过任何关键预警。

## 功能

- **多数据源**: 中国地震台网 (CEIC)、USGS（全球）、中国气象预警
- **震级过滤**: 仅推送高于设定阈值的地震
- **手动查询**: 使用 `!quake list` 命令随时查看最近地震
- **格式化预警**: 按震级颜色标注，显示位置、深度和时间

## 配置

| 设置 | 说明 | 必填 |
|------|------|------|
| 检查间隔 | 轮询间隔（秒），默认 120 | ❌ |
| 最小震级 | 仅推送 ≥ 此震级的地震，默认 3.0 | ❌ |
| 启用 CEIC | 启用中国地震台网，默认开启 | ❌ |
| 启用 USGS | 启用 USGS 全球数据，默认关闭 | ❌ |
| 启用气象预警 | 启用中国气象预警，默认开启 | ❌ |

## 命令

| 命令 | 说明 |
|------|------|
| `!quake list` | 查看最近地震 |
