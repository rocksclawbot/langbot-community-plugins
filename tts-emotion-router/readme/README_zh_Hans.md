# TTS 情绪路由

检测机器人回复中的情绪，自动将 TTS 语音合成路由到匹配的音色和语速。支持任何 OpenAI 兼容的 TTS API。

## 功能

- **情绪检测**: 使用 LLM 或关键词匹配分析回复情绪
- **音色路由**: 将情绪（开心、悲伤、愤怒、兴奋、中性）映射到不同的 TTS 音色
- **语速控制**: 根据情绪自动调整语速
- **灵活的 TTS 后端**: 支持任何 OpenAI 兼容的 TTS API（硅基流动、Azure 等）

## 配置

| 设置 | 说明 | 必填 |
|------|------|------|
| TTS API 地址 | OpenAI 兼容的 TTS 接口地址 | ✅ |
| TTS API 密钥 | API 认证密钥 | ✅ |
| 默认音色 | 中性情绪使用的音色 ID | ✅ |
| 开心/悲伤/愤怒/兴奋音色 | 各情绪对应的音色 ID | ❌ |
| TTS 模型 | 模型名称（默认 tts-1） | ❌ |
| 情绪检测模型 | 用于情绪分类的 LLM 模型 | ❌ |
| 语速倍率（兴奋） | 兴奋情绪的语速倍率 | ❌ |

## 工作原理

1. 机器人发送文本回复
2. 插件检测回复文本的情绪
3. 选择匹配的音色和语速
4. 通过 TTS API 合成语音
5. 发送语音消息

## 支持的情绪

| 情绪 | 默认语速 | 触发示例 |
|------|---------|---------|
| 😊 开心 | 1.0x | 哈哈, 太好了, awesome |
| 😢 悲伤 | 0.9x | 难过, 遗憾, unfortunately |
| 😡 愤怒 | 1.0x | 生气, 烦, damn |
| 🔥 兴奋 | 1.15x | 太棒了, amazing, wow |
| 😐 中性 | 1.0x | （默认） |
